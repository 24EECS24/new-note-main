# 题型3

$\displaystyle 已知函数\ f(\ln x)\ 的定义域为 [1,e]，求\ e^{f(e^x)}\ 的定义域$ 

`````col
````col-md
flexGrow=1
===
==分析：==
$$
\begin{align*}  % *号表示不自动加公式编号，更简洁
&f(\ln x)的定义域为[1,e] \\[4pt]
&即，自变量x\in [1,e]\\[4pt]
&\Rightarrow  \ lnx\in [0,1] \\[4pt]
&即，f(\ \ )的输入范围是[0,1] \\[4pt]
&\because \ e^x的取值范围也是f(\ \ )的输入范围 \\[4pt]
&\therefore \ e^x \in [0,1] \\[4pt]
&\because \ e^x在x\in R上单调 \\[4pt]
&且，e^x=1 \ \Rightarrow \ x=0 \\[4pt]
&\qquad e^x\to 0 \ \Rightarrow \ x\to -\infty   \\[4pt]
&\quad 即，x=0的边界取得到 \\[4pt]
&\therefore \ x\in (-\infty,0] \\[4pt]
\end{align*}
$$

````

````col-md
flexGrow=1
===
==解：==
$$
\begin{align*}  % *号表示不自动加公式编号，更简洁
\because \ &对应法则f(\ \ )相同 \\[4pt]
&则(\ \ )的输入范围相同 \\[4pt]
\therefore \ &y_1=e^{f(e^x)}中e^x的取值范围与 \\[4pt]
&y_2=f(\ln x)中lnx的取值范围相同\\[4pt]
\because \ &由题知y_2(x)的定义域为x\in [1,e] \\[4pt]
\therefore \ &得lnx\in [0,1] \\[4pt]
&则 e^x\in [0,1] \\[4pt]
\therefore \ &\Rightarrow x\in (-\infty,0] \\[4pt]
\end{align*}
$$
`````

`````col
$$
\begin{align*}  % *号表示不自动加公式编号，更简洁
&\because \ \frac{x}{x-1}>0  \\[4pt]
&\therefore \ \frac{x}{x-1}=1+\frac{1}{x-1}>0  \\[4pt]
&\Rightarrow \frac{1}{x-1}>-1  \\[4pt]
&当x-1>0，即x>1  \\[4pt]
&则1>-(x-1)  \\[4pt]
&\Rightarrow x\in (0,+\infty)\cap (1,+\infty)=(1,+\infty)  \\[4pt]
&当x-1<0，即x<1  \\[4pt]
&则1<-(x-1)  \\[4pt]
&\Rightarrow x\in (-\infty,0)\cap (-\infty,1)=(-\infty,0)  \\[4pt]
&综上\Rightarrow x\in (-\infty,0)\cup (1,+\infty)   \\[4pt]
\end{align*}
$$
`````
